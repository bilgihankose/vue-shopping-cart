
# Vuejs Shopping Cart Page

This app created with ```vue cli```
## Run Locally

Clone the project

```bash
  git clone https://github.com/bilgihankose/vue-shopping-cart.git
```

Go to the project directory

```bash
  cd vue-shopping-cart
```
And open with your favorite IDE. And open your terminal.

```
npm install
```

Compiles and hot-reloads for development
```
npm run serve
```

Compiles and minifies for production
```
npm run build
```

Lints and fixes files
```
npm run lint
```
